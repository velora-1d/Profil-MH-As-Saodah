# Miftahul Huda As-Saodah

## Deskripsi
Website profil sekolah Madrasah Ibtidaiyah Miftahul Huda As-Saodah yang mencakup informasi program, fasilitas, berita, prestasi, dan sistem Pendaftaran Peserta Didik Baru (PPDB).

## Stack Teknologi
- Frontend: Next.js 16 (App Router)
- Backend: Next.js Server Actions & API Routes
- Database: PostgreSQL (Neon Serverless)
- ORM: Drizzle ORM
- Auth: Auth.js v5 (Next-Auth)
- Styling: Tailwind CSS, Framer Motion
- Media: Cloudinary
